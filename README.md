# Getting Started
* clone this repo
  * git clone https://github.com/nihamanchem/IITG-Stack-Overflow.git
* Start the server
  * cd server
  * npm install
  * npm start
* Start the client
  * cd client
  * npm install
  * npm start

# Tech Stack
* React.js
* Node.js and Express
* MongoDB
